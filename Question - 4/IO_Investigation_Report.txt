# I/O Investigation Report

## Command 1

**Command Executed:**

```bash
touch logfile.txt
```

**Output:**
File created successfully.

**Observation:**
A log file was created for investigating file access and input/output operations.

---

## Command 2

**Command Executed:**

```bash
tail -f logfile.txt
```

**Output:**

**Observation:**
The command continuously monitors the log file and keeps it open for reading.

**Command Executed:**


## Command 5

**Command Executed:**

```bash
echo "Linux Lab Assignment" > output.txt
```

**Output:**
Data written successfully.

**Observation:**
The output redirection operator (`>`) created the file and stored the command output in it.

---

## Command 6

**Command Executed:**

```bash
echo "Second Line" >> output.txt
```

**Output:**
Data appended successfully.

**Observation:**
The append redirection operator (`>>`) added new content to the existing file without overwriting it.

---

## Command 7

**Command Executed:**

```bash
ls nonexistentfile 2> error.txt
```

**Output:**

**Observation:**
The error message was redirected to `error.txt` using standard error redirection (`2>`).

---

## Command 8

**Command Executed:**

```bash
cat error.txt
```

**Output:**
ls: cannot access 'nonexistentfile': No such file or directory

**Observation:**
The contents of the redirected error file were displayed successfully.

---

## Command 9

**Command Executed:**

```bash
ls > files.txt
```

**Output:**
Directory listing redirected successfully.

**Observation:**
The standard output of the `ls` command was redirected to `files.txt`.
